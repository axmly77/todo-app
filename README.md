 ToDoアプリ

 

 概要

  React + TypeScript で作ったシンプルなタスク管理アプリです。

 

 公開URL

  https://todo-app-blush-rho.vercel.app  



&nbsp;使用技術

&nbsp; - React 18

&nbsp; - TypeScript

&nbsp; - Create React App

&nbsp; 

&nbsp;  機能

&nbsp; - タスクの追加・完了・削除・編集

&nbsp; - 優先度（高・中・低）の設定

&nbsp; - 完了済み / 未完了でフィルタリング

&nbsp; - 進捗バーで達成率を表示
